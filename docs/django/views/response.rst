Http404
-------

.. py:class:: django.http.Http404()

    Исключение генерируется если страница не найдена

    .. code-block:: py

        def index(request, good_id):
            try:
                good = Goods.objects.get(id=good_id)
            except Good.DoesNotExists:
                raise Http404()
            ...
