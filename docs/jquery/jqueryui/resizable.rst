resizable
=========


.. js:function:: resizable([methodName, option, value])
.. js:function:: resizable([methodName, param_obj])
.. js:function:: resizable([param_obj])

    .. code-block:: js
        
        $('#resizable').resizable({
        });


    * `param_obj`

        * `alsoResize` - Селектор, используемый для выбора элементов, размеры которых должны изменяться одновременно с размерами элемента, к которому применено взаимодействие Resizable. Значение по умолчанию — false; оно означает отсутствие таких элементов

        * `aspectRatio` - Если значение этой опции равно true, то изменение размеров элемента будет проис­ходить с сохранением пропорции соотношения сторон. Значение по умолчанию — true

        * `autoHide` - Если значение этой опции равно true, то манипуляторы становятся видимыми лишь тогда, когда указатель мыши располагается над элементом. Значение по умолчанию — false

        * `ghost` - Если значение этой опции равно true, то при изменении размеров элемента будут видны полупрозрачные контуры, отображающие новые размеры элемента. Значение по умолчанию — true

        * `handles` - Определяет, где будут располагаться манипуляторы. 

        * `maxHeight` - Определяет максимальную высоту, до которой можно изменить размеры элемента. Значение по умолчанию — null; оно означает отсутствие ограничений

        * `maxWidth` - Определяет максимальную ширину, до которой можно изменить размеры элемента. Значение по умолчанию — null; оно означает отсутствие ограничений

        * `minHeight` - Определяет минимальную высоту, до которой можно изменить размеры элемента. Зна­чение по умолчанию — null; оно означает отсутствие ограничений

        * `minWidth` - Определяет минимальную высоту, до которой можно изменить размеры элемента. Зна­чение по умолчанию — null; оно означает отсутствие ограничений
