dialog
======

виджет Dialog

.. py:function:: dialog([methodName, option, value])
.. py:function:: dialog([methodName, param_obj])
.. py:function:: dialog([param_obj])

    .. code-block:: html

        <div id="dialog" title="Dialog window">
            Message
        </div>

    .. code-block:: js

        $('#dialog').dialog({
            buttons: [
            {
                text: 'OK',
                click: function(){
                    ...
                }
            }, {
                text: 'CANCEL',
                click: function(){
                    ...
                }
            }]
            position: 'left',
            position: [10, 20],
            position: ['left', 'top'],
            resizable: true,
            title: 'Title',

        });

    * `methodName`

        * `destroy` - Удаляет виджет Dialog из базового элемента

        * `disable` - Отключает диалоговое окно

        * `enable` - Включает диалоговое окно

        * `option` - Изменяет одну или несколько опций

        * `close` - Закрывает диалоговое окно

        * `isOpen` - Возвращает true, если диалоговое окно видимо на экране

        * `moveToTop` - Перемещает диалоговое окно на вершину стека

        * `open` -  Отображает диалоговое окно для пользователя


    * `param_obj`
    
        * `autoOpen` - Если эта опция равна true,
          то диалоговое окно открывается сразу же после его создания
          с помощью метода dialog ().

            Значение по умолчанию — true

        * `beforeClose` - Происходит непосредственно перед закрытием
          диалогового окна.

            Возврат значения false функцией — обработчиком события
            принудительно оставляет диалоговое окно открытым

        * `buttons` - Позволяет указать набор кнопок,
          которые должны быть добавлены в виджет, и функ­ции,
          которые будут вызываться после щелчка на соответствующей кнопке.

            По умол­чанию кнопки отсутствуют

        * `create` - Происходит, когда виджет Dialog применяется к базовому HTML-элементу

        * `close` - Происходит при закрытии диалогового окна

        * `closeOnEscape` - Если эта опция равна true,
          то диалоговое окно можно будет убрать с экрана,
          на­жав клавишу <Esc>.

            Значение по умолчанию — true

        * `drag` - Происходит при каждом перемещении мыши в процессе
          перетаскивания диалогового окна

        * `draggable` - Если эта опция равна true,
          то пользователь сможет перемещать диалоговое окно,
          перетаскивая его заголовок, в пределах окна браузера

        * `dragStart` - Происходит, когда пользователь начинает перетаскивать
          диалоговое окно

        * `dragStop` - Происходит по окончании перетаскивания пользователем
          диалогового окна

        * `focus` - Происходит при получении фокуса диалоговым окном

        * `height` - Определяет начальную высоту диалогового окна в пикселях.

            По умолчанию имеет значение auto,
            при котором высота диалогового окна устанавливается автоматически

        * `hide` - Определяет тип анимации, используемой для сокрытия диалогового окна. 

        * `maxHeight` - Определяет максимальную высоту диалогового окна в пикселях.

            По умолчанию имеет значение false,
            которому соответствует отсутствие ограничений по высоте

        * `maxWidth` - Определяет максимальную ширину диалогового окна в пикселях.

            По умолчанию име­ет значение false,
            которому соответствует отсутствие ограничений по ширине

        * `minHeight` - Определяет минимальную высоту диалогового окна в пикселях.

            По умолчанию имеет значение false,
            которому соответствует отсутствие ограничений по высоте

        * `minwidth` - Определяет минимальную ширину диалогового окна в пикселях.

            По умолчанию имеет значение false,
            которому соответствует отсутствие ограничений по ширине

        * `modal` - Если эта опция равна true,
          то диалоговое окно будет создано как модальное,
          и по­ка оно не будет скрыто,
          пользователь не сможет взаимодействовать с документом

        * `open` - Происходит при открытии диалогового окна

        * `position` - Определяет начальную позицию диалогового окна.

            Значение по умолчанию — center,
            которому соответствует расположение диалогового окна по центру
            окна браузера

            * `bottom`

            * `center`

            * `left`

            * `right`

            * `top`        

        * `resize` - Происходит при каждом перемещении мыши
          в процессе изменения размера диалого­вого окна

        * `resizable` - Если эта опция равна true,
          то диалоговое окно будет иметь кнопку-манипулятор,
          с помощью которой пользователь сможет изменить его размер.

            Значение по умолчанию — true

        * `resizeStart` - Происходит,
          когда пользователь начинает изменять размер диалогового окна

        * `resizeStop` - Происходит по окончании изменения пользователем
          размеров диалогового окна

        * `show` - Определяет тип анимации,
          используемой для отображения диалогового окна.

        * `stack` - Если эта опция равна true,
          то щелчок на диалоговом окне перемещает его на пе­редний план на экране.

            Значение по умолчанию — true

        * `title` - Определяет заголовок диалогового окна

        * `width` - Определяет начальную ширину диалогового окна в пикселях.

            По умолчанию имеет значение auto,
            при котором высота диалогового окна устанавливается автоматиче­ски
