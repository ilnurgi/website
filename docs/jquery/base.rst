.. py:module:: jQuery()

Базовые свойства и методы объекта jQuery
========================================

Данные методы не влия­ют на вы­бор или на вы­бран­ные эле­мен­ты, 
но по­зво­ля­ют об­ра­щать­ся к вы­бран­ным эле­мен­там и вы­пол­нять ите­ра­ции по ним. 

.. py:attribute:: context
    
    Кон­текст, или кор­не­вой эле­мент, в ко­то­ром бу­дет про­из­во­дить­ся вы­бор.


.. py:attribute:: length
    
    Ко­ли­че­ст­во вы­бран­ных эле­мен­тов.


.. py:attribute:: selector
    
    Ори­ги­наль­ная стро­ка се­лек­то­ра


.. py:method:: each(f(index, element))
    
    Вы­зы­ва­ет функцию как ме­тод для ка­ж­до­го вы­бран­но­го эле­мен­та. 
    
    Ос­та­нав­ли­ва­ет ите­ра­ции, как толь­ко функ­ция вер­нет false. 
    
    Воз­вра­ща­ет объ­ект jQuery, от­но­си­тель­но ко­то­ро­го был вы­зван дан­ный ме­тод.


.. py:method:: get(idx)
.. py:method:: get()
    
    Воз­вра­ща­ет вы­бран­ный эле­мент с ука­зан­ным ин­дек­сом в объ­ек­те jQuery. 
    
    Мож­но так­же ис­поль­зо­вать обыч­ный син­так­сис ин­дек­си­ро­ва­ния мас­си­вов с квад­рат­ны­ми скоб­ка­ми. При вы­зо­ве без ар­гу­мен­тов get() дей­ст­ву­ет так же, как toArray().


.. py:method:: index()
.. py:method:: index(sel)
.. py:method:: index(elt)
    
    При вы­зо­ве без ар­гу­мен­тов воз­вра­ща­ет ин­декс пер­во­го вы­бран­но­го эле­мен­та 
    сре­ди смеж­ных с ним эле­мен­тов. 
    
    При вы­зо­ве с се­лек­то­ром воз­вра­ща­ет пер­вый эле­мент из мно­же­ст­ва 
    вы­бран­ных эле­мен­тов, со­от­вет­ст­вую­щий се­лек­то­ру sel, или –1, 
    ес­ли та­кой эле­мент от­сут­ст­ву­ет. 
    
    При вы­зо­ве с эле­мен­том воз­вра­ща­ет ин­декс эле­мен­та elt в мно­же­ст­ве 
    вы­бран­ных эле­мен­тов или –1, 
    ес­ли ука­зан­ный эле­мент не вхо­дит в мно­же­ст­во вы­бран­ных эле­мен­тов.


.. py:method:: is(sel)
    
    Воз­вра­ща­ет true, ес­ли се­лек­то­ру sel со­от­вет­ст­ву­ет хо­тя бы один вы­бран­ный эле­мент.


.. py:method:: size()
    
    Воз­вра­ща­ет зна­че­ние свой­ст­ва length.


.. py:method:: serialize()

    Кодирует набор элементов формы в строку, пригодную для отправки на сервер

    .. code-block:: js
        
        $('form').serialize();


.. py:method:: serializeArray()

    Кодирует набор элементов формы в строку, подготовленную для кодирова­ния в формате JSON

    .. code-block:: js
        
        $('form').serializeArray();


.. py:method:: toArray()
    
    Воз­вра­ща­ет ис­тин­ный мас­сив вы­бран­ных эле­мен­тов.


.. py:method:: unique()

    Сортирует массив объетов HTMLElement в том порядке, 
    в котором они встречаются в документе, 
    и при этом удаляет все дубликаты элементов.
