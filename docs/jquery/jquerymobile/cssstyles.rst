css стили библиотеки
====================

Макетные сетки
--------------

* `ui-grid-a` - 2 столбца
* `ui-grid-b` - 3 столбца
* `ui-grid-c` - 4 столбца
* `ui-grid-d` - 5 столбца


.. code-block:: js
    
    <div class="ui-grid-b">
        <div class="ui-block-a">
            <button>1</button>
        </div>
        <div class="ui-block-b">
            <button>2</button>
        </div>
        <div class="ui-block-c">
            <button>3</button>
        </div>
    </div>
    <div class="ui-grid-a">
        <div class="ui-block-a">
            <button>4</button>
        </div>
        <div class="ui-block-b">
            <button>5</button>
        </div>
    </div>
    <div><button>6</button>/div>
