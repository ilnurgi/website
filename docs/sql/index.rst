Конспекты по SQL
================

SQL - язык программирования для манипулирования и управления базами данных

Группы операторов

   * `DDl` - создающие объекты в базе

   * `DML` - управляющие значениями в таблицах

   * `DCL` - подтверждение прав пользователей
