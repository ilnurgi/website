Единицы измерения
-----------------

* dp, dpi - количество пикселей в одном дюйме,
  позволяет приложениями выглядеть одинаково на различных экранах и разрешениях

    * 252 - 480х800 - 3.7 дюйма

* in – дюйм, определяется по физическому размеру экрана

* mm – миллиметр, определяется по физическому размеру экрана

* pt - 1/72 дюйма

* px - пиксел, не рекомендуется использовать,
  т.к. на разных экранах приложение будет выглядеть по-разному

* sp - тоже что и dp, используется для текстов

Разрешения экранов
------------------

* ldpi - 0.75 dpi

* mdpi - 1 dpi

* tvdpi - 1.33 dpi

* hdpi - 1.5 dpi

* xhdpi - 2dpi
