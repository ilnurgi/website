android.view.ListView
=====================

.. py:class:: android.view.ListView()

    Группа представлений, которая формирует вертикальный список элементов,
    отображая их в виде строк внутри списка.

    Простейший объект ListView использует TextView для вывода на экран значений toString,
    принадлежащих элементам массива.

    Наследник :py:class:`android.view.View`.
