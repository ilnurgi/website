Script - HTML-script
=====================

.. py:class:: Script()


    .. py:attribute:: async

        Име­ет зна­че­ние true, ес­ли эле­мент <script> име­ет ат­ри­бут async, и false – в про­тив­ ном слу­чае. 


    .. py:attribute:: charset

        Ко­ди­ров­ка сим­во­лов в сце­на­рии, на ко­то­рый ссы­ла­ет­ся свой­ст­во src. Обыч­но это свой­ст­во не оп­ре­де­ля­ет­ся, и по умол­ча­нию счи­та­ет­ся, что сце­на­рий име­ет ту же ко­ди­ров­ку, что и вме­щаю­щий его до­ку­мент.


    .. py:attribute:: defer

        Име­ет зна­че­ние true, ес­ли эле­мент <script> име­ет ат­ри­бут defer, и false – в про­тив­ном слу­чае. 


    .. py:attribute:: src

        URL-ад­рес сце­на­рия, ко­то­рый тре­бу­ет­ся за­гру­зить.

    .. py:attribute:: text

        Текст ме­ж­ду те­га­ми <script> и </script>.


    .. py:attribute:: type

        MIME-тип с оп­ре­де­ле­ни­ем язы­ка сце­на­ри­ев. По умол­ча­нию ус­та­нав­ли­ва­ет­ся зна­че­ние «text/ja­va­script», бла­го­да­ря че­му для обыч­ных сце­на­ри­ев на язы­ке Ja­va­Script не тре­бу­ет­ся ус­та­нав­ли­вать это свой­ст­во (или HTML-ат­ри­бут). При при­сваи­ва­нии это­му свой­ст­ву соб­ст­вен­но­го MIME-ти­па мож­но встраи­вать про­из­воль­ные тек­сто­вые дан­ные в эле­мент <script> для ис­поль­зо­ва­ния дру­ги­ми сце­на­рия­ми.
