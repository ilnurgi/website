ImageData - массив пискелей канваса
===================================

.. js:class:: ImageData()


    .. js:attribute:: data
        
        Ссыл­ка, дос­туп­ная толь­ко для чте­ния, на объ­ект, по­доб­ный мас­си­ву, дос­туп­ный для чте­ния и за­пи­си, эле­мен­та­ми ко­то­ро­го яв­ля­ют­ся бай­ты.


    .. js:attribute:: height
        
        Ко­ли­че­ст­во строк пик­се­лов изо­бра­же­ния в мас­си­ве data.


    .. js:attribute:: width
        
        Ко­ли­че­ст­во пик­се­лов в стро­ке изо­бра­же­ния, в мас­си­ве data.
