bool
====

.. py:class:: bool(obj)

    Логический тиn данных.

    Может содержать значения True или False,
    которые ведут себя как числа 1 и 0 соответственно

    .. code-block:: py

        bool(0)
        # False

        bool(1)
        # True

        bool('')
        # False

        bool('cnhjrf')
        # True
