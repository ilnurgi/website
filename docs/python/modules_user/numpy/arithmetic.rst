Арифметические операции
=======================

add()
-----

.. py:function:: add() 

	Сложение, 1 + 1

	.. code-block:: py

		arange(4) + 5
		# array([5, 6, 7, 8])


subtract()
----------

.. py:function:: subtract() 

	Вычитание, 1 - 1

	.. code-block:: py

		arange(4) - 5
		# array([-5, -4, -3, -2])

divide()

floor_divide()

power()

.. py:function:: power()

mod()
-----

.. py:function:: mod()
