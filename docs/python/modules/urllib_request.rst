urlopen()
---------

.. py:method:: urlopen(<url-адрес или объект запроса>[, <POST-данные>][, <тайм-аут>][, cafile=None][, capath=None])

    Выполняет запрос и возвращает результат ответа с сервера
    в виде объекта :py:class:`http.client.HTTPResponse`

    .. code-block:: py

        conn = urlopen(url)
