Псевдоэлементы
==============

after
-----

Добавляет элемент после указанного элемента

.. code-block:: css

    a::after {
        content: ""
    }


before
------

Добавляет элемент до указанного элемента

.. code-block:: css

    a::before {
        content: ""
    }


first-letter
------------

Задает стиль для первого символа в тексте элемента

.. code-block:: css

    p::first-letter {}


first-line
----------

Задает стиль для первой строки текста элемента

.. code-block:: css

    p::first-line {}
