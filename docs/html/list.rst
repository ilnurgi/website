dl
--

Список определений: имя - значение

.. code-block:: html

    <dl>
        <dt>Термин<dt>
        <dd>Описание</dd>
    </dl>


dd
--

Описание термина


ol
--

Нумерованный список

* start - задает начальный номер

* type - тип нумерации

    * 1 - числа

    * a - строчные буквы

    * A - заглавные буквы

    * i - строчные римские цифры

    * I - заглавные римские цифры

.. code-block:: html

    <ol start="5">
        <li>...</li>
        <li>...</li>
    </ol>


ul  
--

.. code-block:: html

    <ul>

    </ul>
