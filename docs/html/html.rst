html
====

Корневой элемент документа.

Все остальные теги документа должны наодится внутри него.

.. code-block:: html

    <!DOCTYPE>

    <html>

       <head></head>

       <body></body>

    </html>


manifest
--------

Ссылка на кеш память

.. code-block:: html

    <html manifest=....>
    </html>


lang
----

Язык содержимого документа

.. code-block:: html

    <html lang='ru'></html>

.. code-block:: html

    <html lang='en'></html>
